Data Extraction & Processing

Ellen Oates

Purpose

  • Chat bot requires an initial data set to be useful
  • WebCMS is difficult to process using pre-built tools
  • Don't want to make the user provide a data set
  • Automatically extracts data given some key pieces of information

Process

  1. User inputs basic information via frontend interface
  2. Input is sent to the server and starts the scraper module
  3. Scrapes web data and recursively follows links to forum posts
  4. Parses data using cheerio by filtering different structures
  5. Analyses the data for keywords using tf-idf algorithm
  6. Data is stored with keywords in the database

Word Bag Model

  • Extract keywords and calculate salience using tf-idf
    • \(tfidf_{i,j} = tf_{i,j} \times idf_{i}\)
    • \(tf_{i,j} = \frac{n_{i,j}}{\sum_{k}^{} n_{k,j}}\)
    • \(idf_{i} = lg \frac{|D|}{|\{j: t_{i} \in d_{j}\}|}\)
